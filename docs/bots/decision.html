🧠 Decision Bot

AI-powered Discord bot that analyzes your decisions against personal values, goals, identity, and experiences using CSV-based RAG system. Provides structured recommendations, reflection questions, and risk-benefit analysis.

🟡 READY FOR DEPLOYMENT 📊 RAG Analysis 🎯 Personal Values 💭 Reflection Questions

🎯 Why This Bot Exists

Making important life decisions can be overwhelming and emotionally charged. The Decision Bot provides objective, personalized analysis by:

  • 🎯 Value Alignment: Analyzes decisions against your personal values and principles
  • 📊 Goal Consistency: Checks if decisions support your short and long-term goals
  • 🧠 Identity Match: Evaluates decisions against your personality traits and identity
  • 📚 Experience Learning: Uses your past experiences to provide relevant insights
  • 💭 Deep Reflection: Asks thought-provoking questions to help you think deeper

📊 Response Format

🎯 ALIGNMENT-ANALYSE

  • Bewertung der Übereinstimmung mit Werten (1-10)
  • Bewertung der Übereinstimmung mit Zielen (1-10)
  • Bewertung der Übereinstimmung mit Identität (1-10)

🧠 DETAILLIERTE BEGRÜNDUNG

  • Aspekte, die zu deinen Werten passen
  • Unterstützung deiner Ziele
  • Widersprüche zu Werten/Zielen

⚡ HANDLUNGSEMPFEHLUNGEN

  • 3-5 konkrete, umsetzbare Schritte
  • Prioritäten für die Entscheidung

💭 REFLEXIONS-FRAGEN

  • Tiefere Fragen zum Nachdenken

⚠️ RISIKO-NUTZEN-BEWERTUNG

  • Positive Auswirkungen
  • Mögliche Herausforderungen

🔧 Technical Stack

🤖 AI & Processing

  • OpenRouter API: DeepSeek model for decision analysis
  • CSV Processing: Automatic data categorization and loading
  • RAG System: Retrieval-Augmented Generation for personalized responses

📊 Data Management

  • CSV Files: Values, goals, identity, experiences
  • Memory Storage: Data loaded and cached for fast access
  • Auto-Detection: Smart column recognition for different CSV formats

💬 Discord Integration

  • Channel ID: 1384282192171110412 (designated decision channel)
  • Error Handling: Graceful handling of missing data and API errors
