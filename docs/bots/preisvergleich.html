🔎 Preisvergleich Bot

Advanced AI-powered deal finder using LangChain and LangGraph. Monitors Notion product watchlist, uses Tavily search API to find current offers, and posts detailed deal notifications every Sunday with comprehensive analysis.

🟢 LIVE & OPERATIONAL 🤖 AI Agent 🔍 Web Search 💰 Deal Tracking

What it does

Notion Watchlist

Reads products from your Notion database.

AI Agent

Uses LangChain/LangGraph and Tavily search to find offers.

Scheduler

Runs every Sunday evening and posts detailed offers.

Short use cases

  • Add an item to Notion → bot searches weekly and notifies about deals.
  • Review posted offers with prices and sources.
