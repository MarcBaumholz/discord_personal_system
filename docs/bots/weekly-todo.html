Interactive • Schedule

Weekly Todo Bot

Creates an interactive daily household tasks list in Discord with emoji reactions and daily reset.

What it does

Interactive List

Displays todos and lets users toggle completion via reactions.

Persistence

Progress saved throughout the day and resets daily.

Commands

✅ to show list, !help_todo for help.

Short use cases

  • Type ✅ to display today's tasks, then mark them via number reactions.
  • Toggle a completed task back to incomplete by reacting again.
