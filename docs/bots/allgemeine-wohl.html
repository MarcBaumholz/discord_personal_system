🏠 Allgemeine Wohl Bot

Intelligent Discord bot that automatically tracks household activities and good deeds, uses AI to categorize activities, maintains statistics, and saves everything to Notion databases including ground truth data.

🟢 LIVE & OPERATIONAL 🤖 AI Categorization 📊 Statistics 💾 Dual Database

🎯 Why This Bot Exists

Tracking household activities manually is tedious and often forgotten. The Allgemeine Wohl Bot solves this by:

  • 🏠 Automatic Tracking: Monitors Discord messages for household activities and good deeds
  • 🤖 AI Categorization: Uses OpenRouter AI to analyze and categorize activities intelligently
  • 📊 Statistics: Maintains comprehensive activity statistics and trends
  • 💾 Dual Storage: Saves to both main database and ground truth reference database
  • ✅ Visual Feedback: Provides reactions and confirmations for user engagement

💾 Database Structure

Allgemeine Wohl Database - Main activity tracking with categorized entries
Ground Truth Database - Reference activities and categories for AI training

🔧 Technical Stack

🤖 AI & Analysis

  • OpenRouter API: AI-powered activity categorization
  • Message Analysis: Real-time Discord message processing
  • Smart Categorization: Intelligent activity classification

💾 Data & Storage

  • Notion API: Dual database integration
