Todoist • Parsing

Todo Bot

Erstellt automatisch Todos aus Channel-Nachrichten, erkennt Priorität/Datum/Labels und integriert sich mit Todoist. Vollständig getestet und stabil im Docker Container.

Funktionen

Automatik

Jede Nachricht → Todo Erstellung mit Smart Parsing.

Smart Parsing

Erkennt Priorität (🔴🟠🟡🟢), Datum und Familienmitglieder automatisch.

Docker Deployment

Läuft stabil im Container mit Health Checks und automatischen Restarts.

Comprehensive Testing

9/9 Tests bestanden, 100% Erfolgsrate, inklusive "piuztzen" Funktionalität.

Commands

!todo, !complete <suche>, !delete <suche>, !stats, !help_todo.

Labels

Marc/Maggie/Gemeinsam oder Username.

Channel: 1368180016785002536

Kurzbeispiele

  • „Wichtig: Einkaufen morgen" → Todo mit hoher Priorität, fällig morgen.
  • „Marc soll Müll rausbringen heute" → Label Marc, fällig heute.
  • „Gemeinsam: Urlaub planen nächste Woche" → Label Familie, nächste Woche.
