🏥 Health Bot

Automated Discord bot that fetches daily Oura Ring health data, analyzes performance against personalized goals, and sends comprehensive health reports with actionable insights every morning at 8 AM.

🟢 LIVE & OPERATIONAL 📊 Daily Reports 🎯 Personalized Goals ⚡ Auto-Scheduling

🎯 Why This Bot Exists

Health tracking without insights is just data. The Health Bot transforms your Oura Ring data into actionable daily guidance by:

  • 📊 Daily Analysis: Automatically analyzes your sleep, activity, and recovery data every morning
  • 🎯 Goal Tracking: Compares your performance against personalized health targets
  • 💡 Smart Insights: Provides 2-3 personalized recommendations based on your data
  • 📈 Progress Monitoring: Tracks trends and identifies patterns in your health journey
  • ⏰ Consistent Routine: Delivers reports at the same time every day for habit formation

🔧 Technical Stack

🏃‍♂️ Health Integration

  • Oura API: Personal Access Token authentication
  • Data Points: Calories, steps, active calories, sleep, recovery
  • Real-time Sync: Daily data fetching and analysis

📊 Analytics Engine

  • Goal Comparison: Performance vs. personalized targets
  • Health Scoring: 4-tier status system (Excellent to Needs Improvement)
  • Trend Analysis: Pattern recognition and insights generation

⏰ Automation

  • Discord.py: Bot framework and message handling
  • Schedule: Daily 8:00 AM report generation
  • Error Handling: Robust API error recovery and logging

💬 Available Commands

!health test Manually generate a health report for testing
!health status Show bot status and current configuration
Auto Daily Automatic daily reports at 8:00 AM (no command needed)

⚙️ Configuration Options

🎯 Goal Settings

  • TARGET_CALORIES: 2200 (default)
  • TARGET_ACTIVE_CALORIES: 450 (default)
  • TARGET_STEPS: 8000 (default)

⏰ Schedule Settings

  • DAILY_SCHEDULE_TIME: 08:00 (24-hour format)
  • TIMEZONE: Europe/Berlin
  • CHANNEL_ID: 1384293986251964527

🔧 API Settings

  • DISCORD_TOKEN: Bot authentication
  • OURA_ACCESS_TOKEN: Oura API access
  • LOG_LEVEL: INFO (configurable)
