⏰ Erinnerungen Bot

Automated Discord bot for daily birthday reminders from Notion database and trash pickup reminders for Schweigheim area. Features dual reminder system with beautiful formatting and timezone awareness.

🟢 LIVE & OPERATIONAL 🎂 Birthday Tracking 🗑️ Trash Calendar ⏰ Daily Automation

Funktionen

Geburtstage

Tägliche Prüfung einer Notion-Datenbank mit formatierten Benachrichtigungen.

Müllkalender

Erinnerung um 20:00 Uhr, wenn morgen Abholung ist (Tonnenarten erkannt).

Befehle

!test_geburtstage, !test_muell.
